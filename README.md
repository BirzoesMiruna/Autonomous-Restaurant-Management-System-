#  Autonomous Restaurant Management System

A comprehensive distributed application designed to automate and optimize restaurant operations. The system features:

* Robust Java Spring Boot Backend: The core logic and data management layer.

* Modern Vue.js Web Dashboard: A dedicated interface for customers to make real-time reservations.

* React Native Mobile Application: A specialized tool for the restaurant manager to monitor and modify tables, waiters, and all active bookings on the go.

* Custom AI Algorithm: Integrated autonomous business logic for system optimization.
---

##  Key Features

* **Full Lifecycle Management (CRUD):** Manage orders and staff with data validation and complex relational mapping in SQL Server.
* **Dual-Platform Interface:**
    * **Web Dashboard (Vue.js 3):** 
    * **Mobile App (React Native):**
* **AI Integration:** Custom-implemented autonomous optimization algorithm

## ⚙️ Installation & Setup Guide

### 1. Database (SQL Server)
1. Open **SQL Server Management Studio (SSMS)**.
2. Create a new empty database named `restaurant_db`.
3. *Note:* Tables are automatically generated by Hibernate upon backend startup (via `ddl-auto: update`).

### 2. Backend (Java Spring Boot)
1. Open the `backend` folder in **IntelliJ IDEA**.
2. Navigate to `src/main/resources/application.properties` and configure your credentials:
   ```properties
   spring.datasource.url=jdbc:sqlserver://localhost:1433;databaseName=restaurant_db;encrypt=true;trustServerCertificate=true
   spring.datasource.username=YOUR_DB_USERNAME
   spring.datasource.password=YOUR_DB_PASSWORD
   
   # Hibernate Configuration
   spring.jpa.hibernate.ddl-auto=update
